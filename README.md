### Dash web application that predicts a patient's risk of mortality based on diagnosis and other metrics available at patient intake.


**Input**:
- Age Group
- Type of Admission
- APR DRG Code
- CCS Procedure Code
- APR Medical Surgical Description
- Payment Typology 1
- Emergency Department Indicator

**Output**

Risk of mortality:
- Minor
- Moderate
- Major
- Extreme
